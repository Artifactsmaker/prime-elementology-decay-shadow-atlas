# Prime Elementology Decay Shadow Atlas

An Operator Intelligence-style HTML dashboard for exploring the Prime Elementology Decay Shadow Atlas dataset.

## Files

- `index.html` - the standalone interactive application.
- `atlas-data.js` - the bundled atlas dataset used by the app.

Keep both files in the same folder.

## How To Open

Open `index.html` in a modern desktop browser. No build step or internet connection is required.

If the browser blocks local script loading in your environment, serve the `outputs` folder with a local static server and open the local URL instead.

## Main Features

- English/Vietnamese language toggle in the header.
- Vietnamese title: `Bản Đồ Bóng Phân Rã Nguyên Tố Học Prime`.
- Interactive Z/N atlas canvas with click-to-select behavior.
- Immediate decay-shadow detail panel beside the atlas viewport.
- Search and filters for grid zone, shadow family, and candidate band.
- Registry table for scanning records.
- Summary metrics and shadow-family distribution charts.

## Data Note

The atlas visualizes simulated rule paths only. It does not imply empirical decay evidence, half-life, or probability.
